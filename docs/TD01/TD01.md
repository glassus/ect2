!!! abstract "Exercice 8"
    === "Énoncé"
        Que fait le code ci-dessous ?
        ```python
        import numpy as np
        import numpy.random as rd

        def mystere(val):
            n = 100
            c = 0
            j = 0
            while j < n:
                x = 1 + np.floor(6*rd.random())
                y = 1 + np.floor(6*rd.random())
                if x + y == val:
                    c += 1
                j += 1

            return c/n
        ```
